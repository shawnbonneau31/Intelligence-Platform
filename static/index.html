Endpoints

GET/api/v1/score?zip=33139Score a single zip code
POST/api/v1/scoreBatch score (Portfolio+)

GET/api/v1/builders?state=FLSearch builders
GET/api/v1/healthHealth check (no auth)

Quick Start

# Score a property curl -H "X-API-Key: YOUR_KEY" \ "http://localhost:8080/api/v1/score?zip=33139" # Batch score (Portfolio tier) curl -X POST -H "X-API-Key: YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{"zip_codes":["33139","77001","85001"]}' \ "http://localhost:8080/api/v1/score" # Download PDF report curl -H "X-API-Key: YOUR_KEY" \ -o report.pdf \ "http://localhost:8080/api/v1/report?zip=33139"
